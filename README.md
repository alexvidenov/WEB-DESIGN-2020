# WEB-DESIGN-2020

#### Repository for the second term WEB Design project in TUES. 

- This static website works as a portfolio for a company, producing electric bike kits (including li-ion batteries, chargers) and offering repairs for pure bikes or only batteries. 
    - It has a minimalistic landing page with popup contact us box, flex-based gallery, service and about page plus one FAQ page with dynamic view height containers.
